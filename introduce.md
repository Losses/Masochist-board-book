# 欢迎使用Masochit-board！

Masochist-board 是一款为现代化浏览器和移动设备量身定制的信息聚合系统（CPS），它如乐高一样，准许您将各种不同的部件拼接在一起，形成一个独一无二的M-b。

您可以把您的Masochist-board打造成您的私人博客、账本或是匿名留言板、论坛。

## 下载 Masochist-board ##

访问 Github 可以下载到我们的测试版本源码。

 - 下载地址：https://github.com/Losses/Masochist-board/releases/tag/v0.1.0
 - 在线演示：https://m-b.science
 - 使用文档：http://losses.gitbooks.io/masochist-board/.

提醒：目前的 Masochist-board 还处在测试阶段，我们建议您不要将其用在生产环境中。
